# SolidWorks and ANSYS Design Portfolio

This repository contains 3D CAD models developed in SolidWorks and validated through ANSYS simulations for structural, dynamic, and performance analysis.

The projects included here demonstrate mechanical design, assembly modeling, and simulation-driven validation workflows.  
Due to file size limitations, raw ANSYS solver result files are not included.
